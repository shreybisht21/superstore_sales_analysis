Superstore Sales Analysis Dashboard

Project Overview
This project analyzes Superstore sales data using Excel, SQL, and Power BI.

The workflow included:
* Data cleaning in Excel
* Data analysis using SQL queries
* Dashboard creation in Power BI

Tools Used
* Excel
* SQL
* Power BI

Data Cleaning
Performed data cleaning and preparation in Excel before importing data into Power BI.

Tasks performed:
* Removed duplicates
* Checked missing values
* Formatted columns
* Prepared dataset for analysis

SQL Analysis
Created SQL queries to analyze business performance.

Queries included:
1. Total Sales
2. Total Profit
3. Sales by Region
4. Profit by Region
5. Top 10 Customers by Sales
6. Top 10 Products by Sales
7. Sales by Category
8. Most Profitable Products

Dashboard Features
* KPI Cards
* Sales Trend Analysis
* Customer Analysis
* Product Analysis
* Interactive Filters

Key Insights
* Identified top-performing products
* Compared sales across regions
* Analyzed customer segments
* Evaluated profitability trends

Files Included
* Raw Dataset
* Cleaned Dataset
* SQL Queries
* Power BI Dashboard
* Dashboard Screenshots
